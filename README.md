# LearnIt - Hack and Wellness 2

> Enabling students to improve their reading skills without the help of a tutor.

## How to Use

**Download**

Download or clone this repo by using the link below:

```
https://github.com/abhishekpatelmc/learnit
```

**Installation**

```
npm install
```

**To Start Server:**

```
npm start
```

**To Vist the App:**

```
http://localhost:3000
```
